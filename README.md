# Microsoft Automation with MS Graph APIs - NestJS Backend
## Description
I'm utilizing NestJS and Microsoft Graph APIs to learn how to automate Microsoft products like MS Teams, SharePoint, and MS Lists.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
